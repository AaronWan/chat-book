Note on Included Data

The directory `data/books/builtin-books.json` distributed with this
Software contains only bibliographic metadata (book titles, author
names, ISBNs, category tags, and brief public descriptions). It does
NOT include full book content, chapter text, or author agent prompt
configurations. Author agent configurations are stored separately
in `data/agents/` and are explicitly excluded from this repository
due to third-party copyright claims on the underlying books.

If you have rights to any book referenced in the project and wish
to contribute its author agent configuration, please contact the
Licensor to arrange a separate contributor agreement.
