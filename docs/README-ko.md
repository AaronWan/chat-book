## 한마디 소개

> **聊书** = 매 대화에서 진정한 사고를沉淀하는 대화형 깊이 독서.

모든 저자는 독자를 주도적으로 사고로 이끌고,碰撞하며, 독창적인洞見을 생성하는 AI 에이전트입니다.

---

## 5분 만에 시작하기

```bash
# 1. 의존성 설치
npm install

# 2. 환경 구성 (.env.example에서 복사)
cp .env.example .env

# 3. 서버 시작
npm start

# 4. 브라우저에서 열기
open http://localhost:3000
```

첫 실행 시 **「+ 책 추가」**를 클릭하여 내장 서고에서 책을 선택하고 대화를 시작하세요.

### 엔드투엔드 테스트 실행

```bash
npm run smoke
```

13개 테스트가 모두 통과해야 합니다.

## 환경 설정

`.env`:

```env
PORT=3000
LLM_URL=https://aihub.firstshare.cn
LLM_KEY=sk-...                          # LLM 키
LLM_MODEL=claude-sonnet-4-6             # OpenAI 호환 모델
DATA_DIR=./data
```

`LLM_KEY`는 필수입니다. 없으면 대화가 LLM을 호출할 수 없습니다.
