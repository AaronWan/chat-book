---

## 一句話定位

> **聊書** = 與作者對話式深度閱讀，把每一場對話中的真實思考沉澱下來。

每個作者是一個 AI 智能體，主動帶著讀者思考、碰撞、生成原創洞見。

---

## 5 分鐘跑起來

```bash
# 1. 安裝依賴
npm install

# 2. 配置環境（已預置 .env.example，直接複製即可）
cp .env.example .env

# 3. 啟動服務
npm start

# 4. 瀏覽器訪問
open http://localhost:3000
```

第一次啟動，先點擊 **「+ 添加書」**，從內置書庫挑一本書，開始聊。

### 運行端到端測試

```bash
npm run smoke
```

預期看到 13 項測試全部通過。

## 配置說明

`.env`:

```env
PORT=3000
LLM_URL=https://aihub.firstshare.cn
LLM_KEY=sk-...                  # 你的 LLM key
LLM_MODEL=claude-sonnet-4-6     # 兼容 OpenAI 協定的模型
DATA_DIR=./data
```

`LLM_KEY` 必填。否則對話無法調用 LLM。
